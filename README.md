# Sales Coach Vinteo

> Coach de vendas com IA treinado em metodologia SPICED · Versão Windows

![Status](https://img.shields.io/badge/status-beta-yellow) ![Plataforma](https://img.shields.io/badge/Windows-10%20%7C%2011-blue)

## O que é

Sales Coach Vinteo é um app desktop que escuta suas calls de vendas em tempo real e sugere a próxima fala consultiva no estilo SPICED (Revenue Architecture / Winning by Design), treinado na metodologia de Jonas Xavier — MVP do Sales Squad da Vinteo.

**Principais recursos:**
- Coach IA em tempo real durante calls (Meet/Zoom/Teams)
- Treinamento com 7 cenários de roleplay (do warming ao Final Boss)
- Biblioteca de scripts SPICED e contornos de objeção
- Analytics das suas sessões salvas

## Download

## Como instalar

1. Baixe o arquivo `Sales-Coach-Vinteo-X.X.X-portable.exe` da última release
2. Dê duplo clique pra rodar (não precisa instalação)
3. No primeiro uso, o Windows pode mostrar "Editor desconhecido" — clique em **Mais informações → Executar mesmo assim**
4. Faça login com sua conta Vinteo em [https://vinteo-sales-coach.lovable.app/](https://vinteo-sales-coach.lovable.app/)
5. Cole sua chave de licença no campo de ativação
6. Pronto pra coachear
